# 💡 Vima PHP Example Applications

This directory contains official reference implementations and integration showcases designed to demonstrate how to use Vima PHP in production applications.

---

## 📂 Available Examples

### 1. [🛒 dp-store](dp-store)
A complete, interactive security playground demonstrating both **Role-Based Access Control (RBAC)** and **Attribute-Based Access Control (ABAC)**. 
- **Framework**: CodeIgniter 4 + Jengo
- **Features Highlighted**:
  - Live simulation switchboard for dynamic context testing.
  - Route guards using `vima_rbac` (role checks) and `vima_authorize` (permission checks).
  - Explicit role/permission freezing (compliance controls).
  - Dynamic ABAC policies evaluating environmental data (IP restrictions, time boundaries, transaction thresholds).
  - Live access audit log viewer.
  
For setup and quick-start instructions, refer directly to the [dp-store README](dp-store/README.md).

---

## 🛠️ Design Philosophy

All examples follow these core principles:
1. **Contract-First**: Operations are built around the interfaces defined in `Vima\Core\Contracts`.
2. **Fail-Closed**: Access is restricted by default; unauthorized states trigger clean 403 pages or JSON responses.
3. **No Magic Strings**: All role and permission checks are fully mapped to class constants generated by Vima.
